# Portfolio Website

A modern, responsive portfolio website template with smooth animations and a clean design. Perfect for showcasing your work, skills, and contact information.

## Features

- Responsive design that works on all devices
- Clean and modern user interface
- Smooth scrolling navigation
- Project showcase section with links
- Skills and technologies section
- Contact form
- Social media links
- Animations on scroll

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome icons

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser to view the website
3. Customize the content in `index.html` with your own information
4. Modify the styles in `styles.css` to match your personal branding
5. Update the contact form functionality in `script.js` to handle actual form submissions

## Customization

### Changing Colors

The main color scheme can be easily modified by changing the color variables in the CSS file:

- Primary color: `#0066cc`
- Secondary color: `#004080`

### Adding Projects

To add more projects to the portfolio, duplicate the project card HTML structure in the projects section and update the content:

```html
<div class="project-card">
  <img src="your-project-image.jpg" alt="Project Name" />
  <h3>Project Name</h3>
  <p>Project description goes here.</p>
  <div class="project-links">
    <a href="https://github.com/your-username/your-repo" target="_blank"
      >View Code <i class="fab fa-github"></i
    ></a>
    <a href="https://your-live-demo-url.com" target="_blank"
      >Live Demo <i class="fas fa-external-link-alt"></i
    ></a>
  </div>
</div>
```

### Updating Skills

Modify the skills lists in the skills section to reflect your own technological expertise.
